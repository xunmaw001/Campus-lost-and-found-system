# Campus-lost-and-found-system
基于Spring Boot和Vue.js的校园失物招领系统旨在为师生提供高效的失物招领服务。用户可以方便地发布遗失物品或找到物品的信息，系统支持物品分类、搜索和信息浏览。通过简洁友好的界面，用户能够快速查找和认领失物，促进校园内的物品归还与交流。该系统提升了校园管理效率，增强了师生之间的互动，营造了和谐的校园环境。
